# AGENTS.md

Este repositório contém uma Codex Skill. Ao modificar a skill:

- Preserve o formato de front matter YAML em `SKILL.md`.
- Não remova as regras centrais de citação e separação entre fatos, inferências e sugestões.
- Não transforme sugestões em fatos recuperados.
- Mantenha a base original em `references/knowledge/`.
- Teste `scripts/search_knowledge.py` com pelo menos uma consulta antes de publicar alterações.
